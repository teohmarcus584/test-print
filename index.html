📘 How to use this test page:
1️⃣ Install one of the bridge apps on your Android tablet:
  • POSBridge – Search Google Play for "POSBridge" (or build from open source)
  • Mobile Print Util – Download from Play Store
2️⃣ Pair your Bluetooth thermal printer with the app (open app → connect printer).
3️⃣ Keep the bridge app running in the background.
4️⃣ Tap the test button above → receipt should print instantly, no dialog.

✅ Success indicator: you'll see a green status message and the physical receipt prints.
❌ If nothing prints, check that the bridge app is open and printer connected.
Tested on Android Chrome / Edge. This page does not use window.print() – prints directly via bridge.
